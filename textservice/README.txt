This sample will take a text string up to 16 characters and display it on the second line of the LCD.

The text service UUID is 9152E0CF-0259-48B7-BA88-261F58D355EA and the text string characteristic 
UUID is D7B00600-78C6-4AEF-93EC-16CE40659C26. You can send up to 16 bytes of ascii hex to this
characteristic and it will be displayed.

For example, write: 31323334353637383930313233343536
This will display: 1234567890123456
